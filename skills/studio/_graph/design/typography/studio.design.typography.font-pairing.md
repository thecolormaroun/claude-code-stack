# Font Pairing

Preferred stack:
- Headlines: geometric sans (Inter, Geist, Satoshi)
- Body: humanist sans or system stack
- Mono: JetBrains Mono, Geist Mono, Fira Code

Principles:
1) Max 2 families (mono is free)
2) Contrast in style, harmony in proportion
3) Prefer variable fonts
4) Consider system fonts for performance
