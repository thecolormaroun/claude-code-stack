# Type Scale

Default tokens (Major Third ~1.25):
- display: 48 / 700 / lh 1.1
- h1: 36 / 700 / lh 1.2
- h2: 28 / 600 / lh 1.25
- h3: 22 / 600 / lh 1.3
- body: 16 / 400 / lh 1.6
- body-sm: 14 / 400 / lh 1.5
- caption: 12 / 500 / lh 1.4

Use with [[studio.design.typography.readability-rules]].
