# Readability Rules

- Body: 16px min, line-height 1.5–1.6
- Measure: ~45–75 chars (target ~65ch)
- Paragraph spacing: ≥ 1em
- WCAG AA minimum (4.5:1 body; 3:1 large)

See also [[studio.design.typography.dark-mode-optics]].
