# Creative Direction (Anti-Generic Visuals)

Studio-native distillation of Dylan Feltus’s `creative-direction` skill.

## Core rules
- **Every image has a job** (hero = emotion, feature = clarity, testimonial = trust).
- **Generic is the enemy**: specificity creates memorability.
- **Consistency beats peak quality**: pick a style and repeat it.

## The Specificity Ladder
1) Generic (“a person at a laptop”) ❌
2) Specific (“a designer reviewing a dashboard on an ultrawide”) ✅
3) Atmospheric (lighting/time/materials) ✅✅
4) Story (props, imperfection, context) ✅✅✅

## Consistency framework
Create a **style prefix** and prepend it to all prompts:
- medium/style (photo/3D/flat)
- palette constraints (dominant + accents)
- perspective
- lighting
- texture/grain

## Avoid these tropes
- glowing orbs, holographic HUDs, purple gradient soup, perfect symmetry

## Practical outputs
- 1 “best prompt” per asset
- 2–3 variations (angle/mood)
- post-processing notes (crop, text overlay, OG constraints)
