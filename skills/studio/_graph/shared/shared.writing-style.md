# Writing Style

- Be concise, but don't omit the crucial rationale.
- Prefer bullets for scans; use short paragraphs for concepts.
- Name patterns explicitly ("Reassurance-first confirmation") so they can be reused.
- Match SOUL.md tone: direct, opinionated, no filler.
- Technical depth: explain the what and why. Keep the how accessible.
