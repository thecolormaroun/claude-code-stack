# User Taste Profile (Template)

**Customize this file with your own preferences.**

## Example Defaults
- Premium, dark, intentional
- Generous whitespace
- Subtle depth (shadows, not flat)
- Purposeful motion
- Avoid corporate SaaS blandness

## Reference Products

## Anti-patterns to Avoid
- Material Design defaults (unless appropriate)
- Bootstrap vibes
- Overly decorative animation
- Information overload without hierarchy
